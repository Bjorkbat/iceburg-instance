Terra

Initialize Terrain

If no terrain data exists, this will create new terrain data for you (read: generates new height data for vertices). If there's already data present, then nothing will happen.

Reset Terrain

This creates entirely new terrain data, regardless of whether or not terrain data actually exists. A nice clean slate

Warning: You'll lose all saved data for the terrain you already have. There's no going back.
